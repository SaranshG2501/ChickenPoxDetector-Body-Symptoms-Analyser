# 📌 About The Project

ChickenPox Detector AI is a healthcare-focused machine learning application developed to assist users with early-stage chickenpox screening through:

- 📷 AI-powered image classification
- 📋 Symptom-based body assessment
- 🤖 Intelligent healthcare workflow integration

The platform analyzes uploaded skin images and combines them with a symptom questionnaire system to generate a preliminary healthcare assessment. The project demonstrates the practical use of Artificial Intelligence in accessible healthcare technology.

# 🧬 System Workflow

## 1️⃣ Image Upload
Users upload an image of the infected skin region through the web application.

## 2️⃣ AI Prediction
The trained machine learning model analyzes the image and predicts possible chickenpox infection patterns.

## 3️⃣ Symptom Evaluation
Users answer a symptom-based questionnaire designed to strengthen preliminary assessment reliability.

## 4️⃣ Final Assessment
The platform combines both image-based and symptom-based results to generate an intelligent healthcare assistance response.

# 🧠 AI Model Overview

The machine learning model is trained on skin infection image datasets to classify visible chickenpox patterns via Roboflow.

- Dataset training
- Pattern recognition
- Prediction confidence generation
